#### insertSort
在有序数列的合适位置插入
```java
public void insertSort(int [] a){
  int len = a.length;
  if(null==a||len<2)
      return 
   for(int i=1;i<len;i++){
      int temp = a[i];
      j=i-1;
      while(j>=0&&a[j]>temp){
          a[j+1] = a[j];
          j--;
      }
      a[j+1] = temp;
   }
}
```
#### bubbleSort
比较交换相邻元素的位置
```java
public void bubbleSort(int [] a){
  if (null == a || a.length < 2) {
		return;
	}
  boolean flag;
  for (int i = 0; i < a.length - 1; i++) {
    flag = false;
    for (int j = 0; j < a.length - 1 - i; j++) {
			if (a[j] > a[j+1]) {
				int temp = a[j];
				a[j] = a[j+1];
				a[j+1] = temp;
				flag = true;
			}
			if (flag == false) {
				return;
			}
		}
  }
  
}
```
#### selectSort
在无序数列中选择一个最小的插入到有序数列的末尾
```java
public void selectSort(int [] a){
  if (null == a || a.length < 2) {
		return;
	}
  for (int i = 0; i < a.length; i++) {
    int k = i;
		for (int j = i + 1; j < a.length; j++) {
			if (a[j] < a[k]) {
				k = j;
			}
		}
		if (k != i) {
			int temp = a[k];
			a[k] = a[i];
			a[i] = temp;
		}
  }
}
```
